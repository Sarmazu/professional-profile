# AGENTS.md

Rules for agents working with this repository:

1. Do not add professional experience, cases, employers, metrics, or achievements unless explicitly asked.
2. Do not invent facts.
3. Do not publish secrets, private data, IPs, tokens, configs, emails, or internal system names.
4. Treat this repository as a public curated output, not as the source of truth.
5. Keep writing concise, factual, neutral, and Markdown-first.
6. Preserve the candidate's authored voice, including vivid, raw, or emotionally colored wording, when it is intentionally present in approved source material.
7. Do not sanitize wording only to make it more corporate or AI-neutral. Edit tone only when explicitly asked or when wording creates a clear public-safety, confidentiality, legal, or factual-risk issue.
8. If unsure whether information is safe to publish, do not include it.
